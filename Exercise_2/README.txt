# Astronaut Daily Schedule Organizer

A Java application to manage the daily schedule of an astronaut, including tasks with different priorities and time slots.

## Features

- Add tasks with start and end times, and priority.
- Remove tasks by description.
- View all tasks sorted by time.
- Mark tasks as completed.
- View tasks by priority.
